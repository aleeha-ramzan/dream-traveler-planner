Dream Traveler Planner is a full-stack travel bucket-list management system designed to help users organize, track, and plan their dream destinations in one secure place. The application allows users to add destinations with details such as country, description, estimated budget, priority level, travel status (Planned/Booked/Visited), and target date. It features a complete authentication system with hashed passwords and JWT-based sessions, role-based access control separating regular Users from an Admin (who can oversee all registered users and destinations through a dedicated Admin Panel), and an automated notification system that alerts users about trips coming up within the next 30 days. Built using MongoDB for data storage, Express.js and Node.js for the backend REST API, and a dark-themed responsive frontend, the project demonstrates practical implementation of NoSQL database design, secure authentication, and full CRUD functionality — with all data verifiable live through MongoDB Compass.
